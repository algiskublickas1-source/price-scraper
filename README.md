\# Price Scraper (Python)



A simple Python CLI application that scrapes product prices from e-commerce websites and stores price history in JSON format.

\- Stores historical price data with timestamps

# Price Scraper (Python)

CLI Python application that scrapes product title and price from a given product URL.

## Features
- Scrapes product name and price
- Uses HTTP headers to avoid blocking
- Handles errors gracefully
- Saves scraped data to JSON file

## Technologies

- BeautifulSoup
